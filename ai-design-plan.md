# Maa's Magic AI Design Plan

Create a fresh, light-themed, fully responsive design plan for Maa's Magic, a Kerala achar brand based in Vadakara. This is a new brand concept from scratch, not a rewrite of any previous mushroom site. The result should be a meticulous plan that can later be used to build a sample `index.html` or an Astro homepage.

## 1. Project Goal

Design a display-first website that helps visitors quickly understand the brand, view the achar varieties, and contact the business through WhatsApp.

Primary goals:

- Present Maa's Magic as a premium homemade achar brand
- Make WhatsApp the main conversion path
- Keep the website light, elegant, and easy to scan
- Make the first release English-only
- Build a design system that can later support Malayalam translation

## 2. Brand Inputs

Use the following requirements from the brief:

- Brand name: Maa's Magic
- Business type: achar / pickle company
- Location: Makkoolpeedika, Vadakara, Kerala
- Sales channel: mainly WhatsApp
- Current scope: display page only
- Framework preference: Astro
- Theme preference: light theme
- Language: English first, translation later if needed

## 3. Logo-Led Visual Direction

The logo suggests a premium, warm, handcrafted identity. The site should feel elegant and food-focused, not corporate or flashy.

### Color Theme

The palette should be built from the logo's black and gold character:

- Primary accent: warm gold
- Secondary accent: deeper antique gold
- Background: ivory or warm cream
- Surface color: soft off-white
- Text: charcoal or near-black
- Borders: muted beige / sand
- Optional support tone: cocoa brown for small highlights

Suggested starting palette:

- Gold: `#C89A2C`
- Deep Gold: `#A97A14`
- Ivory: `#FBF6EA`
- Warm White: `#FFFDF8`
- Charcoal: `#1E1A16`
- Cocoa: `#5B4633`
- Sand Border: `#E7D9BF`

### Visual Mood

The design should feel:

- Premium
- Warm
- Homemade
- Kerala-local
- Trustworthy
- Clean and airy

The design should avoid:

- Dark theme layouts
- Loud neon accents
- Overly generic startup aesthetics
- Heavy gradients
- Busy decorative patterns
- Anything that feels like a mushroom/farm brand

## 4. Content Strategy

The page should be structured around the actual business needs, not filler content.

Core content to show:

- Brand introduction
- Location context
- Product varieties
- WhatsApp ordering
- Trust signals
- Contact details

The initial copy should be in English only. Malayalam can be added later after the layout is approved.

## 5. Required Product List

The homepage should support the achar varieties mentioned in the brief:

- Garlic achar
- Yam achar
- Mango achar
- Beetroot achar
- Prawn achar
- Beef achar
- Chicken achar
- Netholi dry pickle
- Date mix achar

Each product should have a short name, a brief description, and optional future space for size, spice level, or availability.

## 6. Homepage Structure

Design the homepage as a single polished landing page with these sections:

1. Hero section
2. Trust / brand strip
3. Product showcase
4. About / craft / locality section
5. WhatsApp call-to-action section
6. Footer with contact and address

## 7. Hero Section

The hero section should immediately answer three questions:

- What is the brand?
- What does it sell?
- How do I order?

Hero content should include:

- Brand name: Maa's Magic
- Short descriptor: homemade Kerala achar brand
- Location cue: Vadakara, Kerala
- Primary CTA: Order on WhatsApp
- Secondary CTA: View varieties

Hero layout direction:

- Clean two-column layout on desktop
- Stacked layout on mobile
- Strong headline hierarchy
- Large product or logo visual
- Light background with gold visual accents
- Minimal decoration so the page stays premium

## 8. Product Showcase

The product area should be easy to scan and visually refined.

Design the product cards to include:

- Product name
- Short flavor note
- Optional tag such as special, spicy, or bestseller
- Optional space for jar size / batch note / price later

Product card behavior:

- Rounded corners
- Soft shadow or thin border
- Consistent spacing
- Clear visual hierarchy
- Easy to read on mobile

## 9. Trust and Local Identity

Include a short trust section that supports purchase confidence.

Useful trust points:

- Homemade or small-batch preparation
- Vadakara location
- Makkoolpeedika local reference
- WhatsApp ordering convenience
- Freshness / quality messaging only if verified

Keep this section factual and brief. Do not add claims that cannot be confirmed.

## 10. Typography Direction

Typography should look premium and readable.

Recommended direction:

- Headings: refined serif or high-contrast display font
- Body text: clean sans-serif
- CTA text: bold and direct

Typography priorities:

- Strong hierarchy
- Fast readability
- Elegant product branding
- Good mobile legibility

## 11. UI Style Rules

The interface should use:

- Rounded but not cartoonish cards
- Thin warm borders
- Soft shadows
- Gold accents used sparingly
- Clear CTA buttons with strong contrast
- Generous spacing

Avoid:

- Overly playful food graphics
- Dark UI
- Blue/purple startup style
- Excessive animation
- Cluttered section density

## 12. Responsive Behavior

The design must work cleanly across mobile, tablet, and desktop.

Mobile behavior:

- Hero stacks vertically
- Buttons remain easy to tap
- Product cards collapse to one or two columns as needed
- Text remains readable without zooming
- Navigation stays compact

Desktop behavior:

- Hero can use two columns
- Product grid can expand into a balanced layout
- Sections should feel spacious and premium
- Content width should stay readable

## 13. Accessibility Requirements

The design should support:

- Strong text contrast on light backgrounds
- Visible focus states
- Clear button labels
- Logical reading order
- Easy scanning
- Touch-friendly spacing

The page should feel simple and frictionless.

## 14. Implementation Notes for Later

When this plan becomes a sample design and then an Astro build:

- Keep the homepage as the primary landing page
- Put the visual system in shared styles
- Reuse components for product cards and CTAs
- Keep WhatsApp links easy to edit
- Add translation support only after the English version is approved

## 15. Deliverable Definition

The final design plan should be detailed enough that the next step can directly create a clean sample homepage without rethinking the brand direction.

It should be clear that this is a new Maa's Magic identity, built from the logo and the business brief, with no mushroom content carried over.
